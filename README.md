# Cyber Academy

A modern web platform for education and training, built with Next.js and TypeScript.

## Features

### Authentication
- User registration with detailed profile information
- Secure login system
- Protected dashboard routes

### Dashboard
- Personalized welcome dashboard
- Progress tracking through 4 learning levels
- Interactive module navigation
- Profile management

### Learning Modules
- Topic-specific learning paths:
  - Web Development (WebDev)
  - Computer Vision (ComVis)
  - Internet of Things (IoT)
  - Network Security (NetSec)
- Progress tracking per module
- Interactive content delivery

### User Profile
- Comprehensive student information display
- Academic details
- Personal information management
- Topic specialization tracking

### Benefits
- TAK Points system
- E-Certificate upon completion
- CPS (Cyber Physical Systems) insights
- Professional networking opportunities
- Portfolio development
- Knowledge enhancement

### UI/UX Features
- Responsive design
- Animated components using Framer Motion
- Interactive cards with MagicBento
- Staggered animations
- Modern dashboard layout
- Dark theme optimization

## Tech Stack

- Next.js 16
- TypeScript
- TailwindCSS
- GSAP for animations
- Framer Motion
- Radix UI components
- Lottie for animations
- JS-Cookie for auth management
